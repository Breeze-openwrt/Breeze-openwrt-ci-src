## 🚀 本次“去 Node 化”大改造亮点 (性能狂魔特供版)

### 🔴 痛点是什么？
老旧的 Node.js 20 核心即将在 GitHub 停用。原本我们依赖的“历史包清理插件”的作者已经由于各种原因跑路。不论框架层怎么修改 `env` 指令强制降配，只要它调用了遗留依赖包的心跳程序，就会满屏飘红警告，折磨强迫症。

### 🟢 我们做了什么？ (极简架构重构)

为了彻底断绝被第三方依赖卡脖子的隐患，我们直接把那些冗余的旧插件**全部连根拔起**，引入原汁原味的官方 CLI 配合流线型 Shell 脚本，对多余记录实现降维打击。

#### 🛡️ 绝杀招式：超高速历史包自动清理的“流水线”美学
来看一段我们的神级清理代码（无任何附加依赖）：
```bash
  # 列出所有发版记录，跳过最顶端精华的 20 个，剩下的黑名单交由循环轰除
  tags=$(gh release list -L 100 | awk -F '\t' '{print $1}' | tail -n +21)
  for tag in $tags; do gh release delete "$tag" --cleanup-tag -y; done
```
**🧠 编程底层思想大揭秘（Linux 流畅艺术）**：
这短短 3 行实现了前人几千点下载量的冗长第三方组件功能！主要分为三个工段串联（管道符 `|`）：
- **提取员 (`gh release list`)**: 这是 Github 云端环境里默认预装的最佳管家。下达命令后秒速交出版本台账。它使用 Go 语言写成，纯血原生，绝无 Node 引擎报废危机。
- **切割师 (`awk`)**: Linux 的上古文本处理法宝。用一个制表符切开表格，只需取出第一栏的核心版本代号（黑名单猎物）。
- **护城河 (`tail -n +21`)**: 意法叫“从头斩断”。跳过榜单最前的 20 个最新发布（免责金牌），剩下的直接排遣输出结账。最后交由 `for` 循环精准爆破。

这样一番改造后，所有清理阶段的速度将如风驰电掣，永远不看环境变量的脸色！

## ⚡ 究极提速：榨干每一滴服务器性能 (性能狂魔篇)

很多时候，原作者的代码能跑，但在真正的性能极客眼里，“能跑”和“高效”是两码事。我们在 `build-ALL-immortalwrt.yml` 中实施了两次恐怖级的降维提速打击：

### 1. 🔪 屠杀无意义的硬盘 I/O (APT 合并术)
你看之前作者是怎么准备系统环境的？
他复制粘贴了整整 15 次 `sudo apt-get update`！这就是小白典型的做法——什么代码能跑就到处复制一份。这会导致可怜的 GitHub 云端服务器反复扫描了 15 次几十个仓库列表，活生生浪费了服务器 5 分钟的 CPU 和硬盘寿命。

**我们的做法**：大刀阔斧把它砍成了一句话。
- `sudo -E apt-get -qq update`: `# 仅全盘扫描一次！`
- `sudo -E apt-get -qq install -y [所有的几十个包...]`: `# 只派出一辆大卡车把所有货物拉回来！`

### 2. 🧲 架设高维空间传送门 (`actions/cache`)
原本编译 OpenWrt 时，有一个致命的 `make download` 环节。这会要求 GitHub 跨越太平洋，去源站下载所有 C 语言基础模块包，一旦网络抽风直接完蛋。

**神级技术介入第一层 (网络护城河 `dl`)**：
```yaml
uses: actions/cache@v5
with:
  path: openwrt/dl
```
这就是闻名遐迩的**云端快门缓存拦截网**。
第一次跑流水线：我们会乖乖网购（下载几个 G 的包裹）。跑完后，拦截网把这些包裹统统封存进云端秘密基地。
第二次跑流水线（可能是一年后）：只要网购清单不变，这层拦截网会大喊一声“且慢，我有备用货！” 然后瞬间把几个 GB 的包裹砸在运行机器上。

**神级技术介入第二层 (碎片金库 `ccache`)**：
如果你修改了某一个路由器小程序的代码，是不是连带它附带的成千上万个不需要更改的 Linux 底层核心库，也被迫跟着重装了一遍？
为了杜绝这种蠢事，我们在流水线植入了高级编译代理：
1. 我们让你的 `use_ccache` 菜单勾选框生效，在执行 `make defconfig` 前，强行给它的内脏贴上一张符箓：`echo "CONFIG_CCACHE=y" >> .config`。
2. 配置好 `actions/cache@v5` 接管 `~/.ccache`（对象缓存库）。
这就像是买了一套拼图，如果不小心弄丢了一块，以前系统是用刀把木头重劈一次做个全新的地图。现在？系统扫一眼金库，直接拿出了那 999 块现成拼图糊了上去。只劈 1 块。时间从 3 小时直降 30 分钟！

**这，就叫极致双轨缓存。**
